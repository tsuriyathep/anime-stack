ANIME Stack
Skeleton project for (A)ngularJS, (N)odeJS, (I)cedCoffeeScript, (M)ongoDB, (E)xpress.
  1. ALL server AND client code is (Iced) Coffee Script including Grunt!
  2. Running "grunt dev" will watch and compile client coffee scripts.
  3. Running "grunt prod" will do merge all coffee scripts to one minified JS file.
  4. Running "grunt heroku" will do 'prod' then push to Heroku.
  5. REST examples with MongoDB import JSON file, and some other mongo functions.
REST API = See /server/server.coffee
  • /db/reset - Empty mongo collection
  • /db/import - Add some data from JSON file to mongo collection
  • /db/list - List data in mongo collection
